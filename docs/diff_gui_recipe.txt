data selection
 - cluster dropdown list selection
   - change cluster, ok, network, station, stream, channel, profile.raw, profile.ok
 - snr text value
   - change ok, network, station, stream, channel, profile.raw, profile.ok
 - network dropdown
   - change ok, network, station, stream, channel, profile.raw, profile.ok
 - station dropdown
   - change ok, station, stream, channel, profile.raw, profile.ok
 - stream dropdown
   - change ok, stream, channel, profile.raw, profile.ok
 - channel dropdown
   - change ok, channel, profile.raw, profile.ok
 - distance range
   - change ok, distmax, distmin, profile.raw, profile.ok
 - azimuth range
   - change ok, azmax, azmin, profile.raw, profile.ok
